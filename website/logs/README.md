# Логи приложения

Логи системы FlowerShop сохраняются в этой папке.

## Файлы

- `flowershop.log` - логи покупок и операций с товарами

## Формат логов

```
2026-01-14 20:42:59 - НАЧАЛЬНОЕ КОЛИЧЕСТВО ТОВАРОВ:
  - Розовые розы: 10 шт.
  - Подсолнухи: 8 шт.
  ...

2026-01-14 23:09:19 - ПОКУПКА: Клиент Иван (ivan@mail.ru) купил 2 шт. товара 'Подсолнухи' по цене $39.99. Было: 8 шт., осталось: 6 шт.
```

## В Docker

Логи монтируются из контейнера backend:
- Контейнер: `/app/logs/flowershop.log`
- Хост: `./logs/flowershop.log`
